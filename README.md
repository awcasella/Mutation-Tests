# Mutation-Tests
Example of how to install and use mutation tests

# Backend
    $ mvn org.pitest:pitest-maven:mutationCoverage

# Frontend
    $ npx create-react-app mutation
    $ npm start
    $ npm - striker-cli
    $ npm i --save-dev @stryker-mutator/jest-runner
    $ npx stryker init
    $ npx stryker run
